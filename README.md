# helpDesk
Site responsivo de Help Desk. Tecnologias: HTML, CSS, Bootstrap e PHP.

Features:
- Login e Autenticação
- Abrir chamados
- Consultar chamados
- Controlar visualização por meio de tipo de usuário adm ou usuario
